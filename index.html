Generic Markup and CSS

"cdb-" is the namespace that we are going to use for all the components. We do this for avoid conflicts with other libraries which could use similar class names

  • HTML

              
							/* Module name (we need for js) */
							<div class="module-name">
							  /* cdb-name == component name */
							  <div class="cartodb-component cdb-name">
							    Content
							 	</div>
							</div>
